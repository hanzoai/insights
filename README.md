# Hanzo Insights

Product analytics and feature flags.

## Overview

Hanzo Insights is a comprehensive product analytics platform with feature flags, session recording, and A/B testing. Understand how users interact with your AI products and ship features with confidence.

## Features

- **Event Analytics**: Track user actions and funnels
- **Feature Flags**: Progressive rollouts and experiments
- **Session Recording**: Replay user sessions
- **A/B Testing**: Statistical experiment analysis
- **Cohorts**: Group users by behavior
- **Self-hosted**: Full data ownership

## Quick Start

```bash
docker run -p 8000:8000 hanzo/insights
```
